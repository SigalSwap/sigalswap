## 2. Description of Services

The Interface provides a web-based means of interacting with the SigalSwap
Protocol, a set of autonomous smart contracts deployed on the Aztec Network
(a Layer 2 blockchain built on Ethereum).

The Interface allows Users to:
- Swap digital assets through automated market maker (AMM) liquidity pools
- Provide liquidity to pools and receive liquidity provider (LP) tokens
- Remove liquidity and redeem LP tokens for underlying assets
- View pool information, price quotes, and position values

**The Interface is distinct from the Protocol.** The Protocol consists of
autonomous, immutable smart contracts on the Aztec Network that operate
independently of the Company. The Company does not control, operate, or
administer the Protocol. Anyone can interact with the Protocol through
